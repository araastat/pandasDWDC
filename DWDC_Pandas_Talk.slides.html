Snake eating data¶

Most data sets we see are mixed type, i.e. mix of numerical, categorical and string

This was a PAIN to deal with (it still is in Matlab)

Snakes eating data¶

Pandas is a Python data analysis package developed by Wes McKinney in 2008

  • Currently at version 0.16
  • Well-documented
  • http://pandas.pydata.org

Why use pandas?¶

  • Created right kinds of data structures

    • Mixed type container (DataFrame)!!
    • Indexed data structures (refer by name or position)

  • Convenient

    • Nice wrappers around most functionality
    • Takes advantage of chained operations

Why not use pandas?¶

  • Convenient!!

    • Not necessarily most efficient for production
    • Loads all data into memory
    • Requires long time to compile, for use in virtualenv

Nice blog by Ben Bengfort on reading CSV in Python efficiently
